# DarkFimoz Shaders

Realistic shader pack for Minecraft 1.20-1.21.11 with OptiFine and Iris support.

## Features

- **Player & Entity Shadows** - Dynamic shadow casting from all entities
- **Motion Blur** - Smooth camera movement effect
- **Realistic Sky** - Dynamic day/night sky with atmospheric scattering
- **Realistic Sun** - Enhanced sun rendering with proper lighting
- **Realistic Moon** - Improved moon appearance and glow

## Installation

### OptiFine
1. Download OptiFine for Minecraft 1.20-1.21.11
2. Download shader pack from [Releases](../../releases)
3. Place `.zip` file in `.minecraft/shaderpacks/`
4. Select in Options → Video Settings → Shaders

### Iris
1. Install Fabric and Iris mod
2. Download shader pack from [Releases](../../releases)
3. Place `.zip` file in `.minecraft/shaderpacks/`
4. Select in Options → Video Settings → Shader Packs

## Settings

- **Shadow Distance** - Shadow render distance (default: 128)
- **Motion Blur Strength** - Motion blur intensity (default: 0.5)
- **Sky Brightness** - Sky lighting level (default: 1.0)
- **Sun/Moon Size** - Celestial body size multiplier (default: 1.0)
